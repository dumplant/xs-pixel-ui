# Start

### 快速开始

#### 1. 安装组件库

```
npm i xs-pixel-ui
```

#### 2. 引用组件库

```javascript
// 全部引入
import 'xs-pixel-ui/dist/css/index.css';
import PUI from 'xs-pixel-ui';
Vue.use(PUI);

//按需引用
import 'xs-pixel-ui/dist/css/components/button.css';
import PixelButton from 'xs-pixel-ui';
Vue.use(PixelButton)

```
